# Chisel-template
Your next Chisel project template

## Prerequesties 
1. Java 11 JRE and SDK 
2. SBT tool [link](https://www.scala-sbt.org/download.html) 

## Usage 

1. Instantiate the template
```bash
> mkdir my-project
> cd my-project 
> sbt new Neurodyne/chisel-template.g8
```

2. Run tests locally 
```bash
> sbt 
> test
```

3. Add your code
